================================================================================
CZĘŚĆ 1: PRZYGOTOWANIE
================================================================================

KROK 1: Przejdź do Report view
  Po lewej: Report (lub Ctrl+Alt+1)

KROK 2: Zmień nazwę strony
  Kliknij prawo na tab "Page 1" (u dołu)
  Rename page
  Wpisz: "Sprzedaż: USD vs PLN"
  Enter

KROK 3: Dodaj tytuł
  Insert → Text Box
  Narysuj na górze
  Wpisz: "Analiza Sprzedaży: USD vs PLN (Rok 2023-2025)"

  Formatowanie:
  - Font size: 24
  - Bold: ☑
  - Color: Navy Blue


================================================================================
CZĘŚĆ 2: PIERWSZY WYKRES (Column Chart)
================================================================================

TYTUŁ: "Sprzedaż USD vs PLN wg Daty"

KROK 1: Wstaw Column Chart
  Insert → Column Chart (Clustered Column Chart)
  Narysuj pod tytułem (zajmuje lewą połowę ekranu)

KROK 2: Dodaj oś X (daty)
  W panelu Visualizations (po prawej):

  Sekcja: Axis
  Przeciągnij: 'extract sales_order_header'[order_date] do pola Axis

  Zmień agregację:
  - Kliknij na order_date w Axis
  - Wybierz: Day (nie Month, Year - Day!)
  - Lub: Date (nie Date Hierarchy)

KROK 3: Dodaj Values Y (sprzedaż)
  Sekcja: Values
  Przeciągnij: SalesUSD do pola Values
  Przeciągnij: SalesPLN do pola Values

  Wynik: Dwie serie na jednym wykresie

KROK 4: Formatuj wykres
  Po zaznaczeniu wykresu:

  Format visual (u dołu panelu Visualizations):

  a) Title:
     ☑ On
     Title text: "Sprzedaż USD vs PLN (Lata 2023-2025)"
     Font size: 16
     Alignment: Center

  b) Legend:
     ☑ On
     Position: Right
     Title: "Waluta"

  c) X-axis:
     ☑ Title
     Title text: "Data zamówienia"
     Font size: 12

  d) Y-axis:
     ☑ Title
     Title text: "Wartość sprzedaży (USD/PLN)"
     Font size: 12

  e) Colors:
     SalesUSD: Niebieski (#0078D4)
     SalesPLN: Zielony (#107C10)

WYNIK: Piękny Column Chart z dwiema kolorowymi kolumnami ✓


================================================================================
CZĘŚĆ 3: DRUGI WYKRES (Combo Chart - Sprzedaż vs Trend Kursu)
================================================================================

TYTUŁ: "Sprzedaż vs Trend Kursu Walut"

KROK 1: Dodaj nową stronę
  Kliknij prawo na tab "Sprzedaż: USD vs PLN"
  New page
  Zmień nazwę: "Sprzedaż vs Trend Kursu"

KROK 2: Wstaw drugi Column Chart
  Insert → Combo Chart
  (Jeśli brakuje - użyj Line Chart zamiast)
  Narysuj na środku

KROK 3: Dodaj dane do wykresu
  Visualizations - Axis:
  Przeciągnij: 'extract sales_order_header'[order_date]

  Visualizations - Column Y-axis:
  Przeciągnij: SalesUSD

  Visualizations - Line Y-axis:
  Przeciągnij: AvgRate

  (Jeśli brakuje Line Y-axis - używasz Column Chart, nie Combo)

KROK 4: Jeśli nie ma Line Chart - użyj drugiego Column Chart
  Alternatywa:

  Insert → Line Chart
  X-axis: order_date
  Y-axis: SalesUSD
  Y-axis: AvgRate (druga seria)

KROK 5: Formatuj wykres
  Format visual:

  a) Title:
     ☑ On
     "Sprzedaż USD vs Trend Kursu USD/PLN (Lata 2023-2025)"

  b) Legend:
     ☑ On
     Position: Bottom
     Labels:
     - SalesUSD: "Sprzedaż (USD)"
     - AvgRate: "Kurs USD/PLN"

  c) X-axis Title:
     ☑ On
     "Data"

  d) Y-axis (Column):
     ☑ Title
     "Wartość sprzedaży (USD)"

  e) Y-axis (Line):
     ☑ Title
     "Kurs USD/PLN"

  f) Colors:
     Sprzedaż: Niebieski
     Kurs: Pomarańczowy/Czerwony

WYNIK: Combo chart pokazujący dwie osie Y ✓


================================================================================
CZĘŚĆ 4: DODAJ SLICER (FILTR DATY)
================================================================================

KROK 1: Na OBIE strony dodaj Slicer
  Insert → Slicer
  Narysuj na górze (nad wykresami)

KROK 2: Wybierz pole do filtrowania
  Pojawi się: Select a field
  Rozwiń: 'extract sales_order_header'
  Wybierz: order_date
  Kliknij: Add

KROK 3: Skonfiguruj Slicer
  Po zaznaczeniu Slicera:

  Format visual:
  - Type: Between (Zakres)
  - Style: Dropdown (lub List)

  Zaznacz daty:
  Od: 2023-01-01
  Do: 2025-06-30

  ☑ Require every visual has this filter
  (Żeby slicer filtrował wszystkie wykresy)

WYNIK: Slider pokazujący daty - możesz zmieniać zakres 👍


================================================================================
CZĘŚĆ 5: DODAJ KARTY PODSUMOWUJĄCE
================================================================================

KROK 1: Dodaj Card na pierwszej stronie
  Insert → Card (Multi-row Card)
  Narysuj po prawej stronie od wykresu

KROK 2: Dodaj miary do kart

  Card 1:
  Fields: SalesUSD
  Pojawi się: ~12,000,000 (suma)
  Format: Currency
  Title: "Łączna sprzedaż USD"

  Card 2:
  Fields: SalesPLN
  Pojawi się: ~48,000,000
  Format: Currency
  Title: "Łączna sprzedaż PLN"

  Card 3:
  Fields: TotalSalesCount
  Pojawi się: 121,317
  Format: Number
  Title: "Liczba transakcji"

  Card 4:
  Fields: AvgRate
  Pojawi się: 4.0539
  Format: Decimal
  Title: "Średni kurs USD/PLN"

KROK 3: Formatuj karty
  - Font size: 18
  - Background: Jasny szary
  - Alignment: Center


================================================================================
CZĘŚĆ 6: DODAJ TEKST OBJAŚNIAJĄCY
================================================================================

KROK 1: Na drugiej stronie dodaj Text Box
  Insert → Text Box
  Narysuj pod wykresem

KROK 2: Wpisz objaśnienie
  "INTERPRETACJA:
  • Niebieskie kolumny = Sprzedaż (USD)
  • Pomarańczowa linia = Kurs USD/PLN
  • Wzrost linii = PLN słabnie (USD drożeje)
  • Spadek linii = PLN umacnia się (USD taniej)
  • Obserwacja: Czy sprzedaż rośnie gdy kurs rośnie?
    - TAK = wyższa sprzedaż przy słabym PLN
    - NIE = wyższa sprzedaż przy silnym PLN"

WYNIK: Tekst objaśniający do interpretacji 📝
